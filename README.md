# bash-scripts

Kubernetes development environment
=======

This script sets up the following:

* kubectl (and kubectl aliases)
* kubens and kubectx
* kubespy
* GnuPG
* Sops
* Velero
* Helm
* Helmfile
* kubeval

AWS CLI 2
=======

This script sets up the following:
* AWS CLI
* AWS IAM access configuration
